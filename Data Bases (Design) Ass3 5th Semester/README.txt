δόθηκαν...
Η δομή του κώδικα είναι μόνο οι τέσσερις συναρτήσεις που μας είχατε δώσει. Δεν έχει χρησιμοποιηθεί καμία άλλη συνάρτηση.
Γενικά ότι λεπτομέρεια υπάρχει στον κώδικα έχει γραφεί πολύ αναλυτικά στην αγγλική γλώσσα σε σχόλια εντός του κώδικα..
Τα μόνα επιπλέον σχόλια που πρέπει να προσθέσω για την κάθε συνάρτηση είναι τα εξής:
1. Η πρώτη και η δεύτερη συνάρτηση είναι τόσο εύκολο να δούμε τι κάνει που δεν χρειάζονται κάποιο επιπλεόν σχόλιο πέραν αυτών που υπάρχουν ήδη εντός του αρχείου app.py.
2. H τρίτη συνάρτηση μετά την εκτέλεση του query επιστρέφει όλες τις αρρώστιες που εμφανίστηκαν σε όλα τα νοσοκομεία ξεχωριστά για το καθένα σε φθίνουσα σειρά του αριθμού εμφάνισης.
   Επειδή όμως εμείς ζητάμε να ενφανίζονται μόνο οι πρώτες 'n' αρρώστιες κατά σειρά εμφάνισης ουσιαστικά στον python κώδικά μας ελέγχουμε για κάθε νοσοκομείο να εμφανίζονται οι γραμμές
   που είναι <= 'n'  
3. H τέταρτη συνάρτηση μετά την εκτέλεση του query επιστρέφει όλα τα άμεσα ΄κολλήματα΄... Οπότε αυτό που κάνουμε στον κώδικά μας python είναι να βρίσκουμε και όλα τα έμμεσα για κάθε 
   ασθενή που 'προσβλήθηκε' από την αρρώστια. Για να το κάνουμε αυτό οι παραδοχές που χρησιμοποιήσαμε είναι:
   α. ο pname_from_2 που κόλλησε τον pname_to_2 να έχει και ο ίδιος κολλήσει από ένα pname_from_1 δηλαδή σε κάποιο άλλο αποτέλεσμα που επέστρεψε το query ο pname_from_2 να ήταν ο pname_to_1.(νομίζω έγινα κατανοητός)
   β. επιπλέον ο pname_to_2 να έχει κολλήσει την ασθένεια μετά τον pname_from_2 που ήταν κάποτε pname_to_1..(προφανώς!) Αν αφαιρέσουμε αυτό τον έλεγχο δείτε τα καινούργια αποτελέσματα του αλγορίθμου για την ασθένεια Epilipsy για παράδειγμα.
      Αυτά έχουν :
            ..............
            Maria Newton   | Robert Depp  | Maimonides   | 1911/12/01 |Harlem         | 1935/06/01
            Enrique Fermat | Maria Newton | Papanikolaou | 1937/03/02 |Western Attica | 1983/01/01
            ..............
      -->   Enrique Fermat | Robert Depp  | Maimonides   | 1911/12/01 | Bixby         | 1942/05/02

      που όμως δεν γίνεται να ισχύουν γιατί ο Robert Depp κόλλησε τον 1911/12/01 ενώ Enrique Fermat κόλλησε την Maria Newton το 1937/03/02 η οποία μετά ξανακόλλησε τον Robert Depp για δεύτερη φορά με την ίδια ασθένεια, αλλά αυτό δεν έχει 
      ως επακόλουθο αυτό που ζητάμε
    γ. Τέλος κάνουμε έλεγχο για τα duplicates... Με τον επαναληπτικό τρόπο που ορίσαμε τον αλγόριθμο προκύπτουν αρκετά duplicates τα οποία δεν πρέπει να τα κάνουμε append στα αποτελέμστα που εμφανίζουμε
